# PRODIGY_WD_02

This project is a custom stopwatch web application developed using HTML, CSS, and JavaScript. It provides users with functionalities to accurately measure time intervals and record laps.

## Features

 **Timer Display:** Shows time in hours, minutes, seconds, and milliseconds.<br>
 **Start/Pause Button:** Controls the start and pause of the stopwatch.<br>
 **Reset Button:** Resets the stopwatch to zero.<br>
 **Lap Button:** Records and displays lap times during stopwatch operation.<br>
 **Responsive Design:** Ensures compatibility and optimal display on various devices.

## Technologies Used

 **HTML:** Used for structuring the elements of the application.<br>
 **CSS:** Stylesheet for layout, design, and animations.<br>
 **JavaScript:** Enables dynamic functionality such as time calculations and event handling.
